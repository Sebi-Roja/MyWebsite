Personal Project

Arduino Halloween Light Show

Role Designer & Builder
Tools Arduino, C++
Hardware LEDs
Duration October 2024

Overview

I built a synchronized Halloween light display with an Arduino controlling several LEDs to activate in sync with a note played from a speaker, so that the lights could change along with the music, to create a show.

How it works

After writing the Arduino C++ code and uploading it to the Arduino, the entire circuit was placed in a box so that it could be discovered. The circuit was designed with a photoresistor to measure light exposure, so that it would be aware whether the box had been opened, and if so, how much. This would determine which frequencies to play, and which pattern of lights to display. The notes were picked out, and their frequencies given to the speaker to play, along with how long they should be held for, and which lights should be on for that corresponding time duration.
